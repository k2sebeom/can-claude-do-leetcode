# Instruction

Read problem under ./problems/ and write your solution under ./solutions.

# Solution Format

Write in .md file with the same filename as the problem.

Solution document should be:

Code: in code block
Chain of Thought: Imagine you are a test applicant seeing this question for the first time. Describe the chain of thought in finding the answer to this code challenge. Make it like a real process that applicant solves during the interview. At each step of thought, include the code that is written and considerations.
Complexity: Details on time complexity of your solution.
Algorithms: Notable algorithms that are related.
